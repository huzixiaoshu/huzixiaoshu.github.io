Vim编译器使用

Created2021-03-30|Updated2021-03-30
|Post View:

基础学习

  1. echo+”helo wordl” 终端显示字符
  2. $从变量中取值

    Vim 编译器的使用

  3. 工作模式：
  • 命令模式：打开文件之后，默认进入命令模式
  • 编辑模式：需要输入一些命令，切换到编辑模式
  • 末行模式：在末行模式下可以输入一些命令
  1. vi操作：
  • h向左，j向下，k向上，l向右
  • 0 移动到行首，$移动到行尾
  • gg移动到文本首，G行尾，300G第300行
  • x删除光标所在的字符，u撤销，X删除前面的字符。
  • dw删除单词，光标所在到后面的单词。
  • d0删除到行首，D或d$删除行尾
  • dd删除当前行（实质是剪贴），删除多行:ndd(n–自然数)
  • ctrl + r反撤销
  • p从当前行的下一行开始粘贴
  • P从当前行开始粘贴
  • yy复制一行，粘贴是p/P,复制多行nyy
  • v可视模式，可视模式下复制没有换行，复制y, 删除d 移动光标hjkl
  • 查找：
    • /stdio.h 从光标位置向下查找
    • ?stdio.h 从光标位置向上查找
    • # 光标移动到单词上，然后查找
    • 遍历的快捷键 N/n
  • r 替换字符，只能是单个
  • 缩进：向右：>>，向左<<
  • 查看man文档，并指定第几章：3 shift + k
  • a当前字符后开始插入，A当前行的末尾
  • i当前字符前开始插入，I当前行的首端
  • o当前行的下一行开始，O当前行的上一行开始
  • s删除光标后面的字符然后开始插入，S删除光标所在行开始插入
  • 输入 : 进入末行模式
  • : s/tom/jack 光标所在行第一个tom替换成jack
  • : s/tom/jack/g 光标所在行所有tom替换成jack
  • : %s/tom/jack/g 文本内所有的tom变成jack
  • : 27，30s/tom/jack/g 27行到30行所有tom替换成jack
  • : !+命令 在末行模式下执行一些命令
  • w保存文件，q退出，wq保存退出，q!不保存退出。
  • x== wq
  • 命令行下退出ZZ
  • 分屏操作：
    • sp将屏幕分为两个部分–水平
    • vsp 将屏幕分为两个部分–垂直
    • sp(vsp) + 文件名 水平或垂直拆分窗口显示两个不同的文件
    • 命令：wqall保存并退出所有屏幕
    • 命令：wq保存并退出光标所在的屏幕
    • Ctrl+ww 切换两个屏幕
