linux 基础学习2

Created2021-03-24|Updated2021-03-25
|Post View:

网络相关命令

  1. ifconfig 查看ip相关信息
  2. ping + ip ping 如果不按ctrl+c就一直ping下去，加上-c 4 则是ping了4次之后停止。
  3. nslookup + 网址 可以查看网址的ip地址。

    增加用户

  4. sudo adduser *** 增加新的用户 ，如果有大写的则报错
  5. su *** 切换用户
  6. sudo useradd -s /bin/bash -g Robin -d /home/Robin -m Robin:可以创建一个大写的名字的用户
  • 如果报错则 sudo groupadd Robin
  • sudo passwd Robin 指定密码
  1. sudo passwd root 修改root密码
  2. sudo deluser +名字删除用户
  • sudo userdel -r +名字删除用户下所有的目录文件

服务器安装

ftp服务器搭建 -vsftpd

用途：负责文件的上传和下载

  • 首先sudo apt-get install vsftpd安装
  • 修改配置文件 - 如何修改配置文件 – /etc/vsftpd.comf
  • sudo service vsftpd restart 重启服务器

    ftp客户端

  • 实名用户登录
    • ftp+ IP (server)
    • 输入用户名(server)
    • 输入密码
  • 文件的上传与下载
    • 上传：put file
    • 下载：get file
    • 不允许操作目录，如果想操作目录 –打包 tar/rar/zip
  • 匿名登录
    • ftp +serverIP
    • 用户名：anonymous
    • 密码： 直接回车
    • 不允许在任意目录直接切换
    • 只能在一个指定的目录直接切换
    • 需要在ftp服务器上创建一个匿名用户的目录-【匿名用户的根目录】

      nfs服务器搭建：

  • 服务器端：
    • 创建共享目录：mkdir dir
    • 修改配置文件:
      • /etc/exports
      • /home/itcast/share *(ro,sync)
    • 重启服务： sudo service nfs-kernel-server restart
  • 客户端:
  • 挂载服务器共享目录：mount serverIP:sharedir /mnt

    ssh服务器：

  • 安装ssh: sudo apt-get install openssh-server
  • 查看ssh是否安装 ： sudo aptitude show openssh-server
  • 远程登陆：ssh 用户名@IP:确定连接的时候一定要写yes/no
  • 退出登录：logout

    公网ip获取

  • curl ifconfig.me
  • 内网： ifconfig
