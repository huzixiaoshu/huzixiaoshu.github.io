gdb调试

  1. 启动gdb
  • start –只执行一步
  • n –next
  • s –step（单步） –可以进入函数体内部
  • c –continue 直接停在断点位置
  1. 查看代码
  • l –list
    • l 10
    • l filename ：行号（函数名）
  1. 设置断点：
  • 设置当前文件断点：
    • b 10 (函数名)
    • b fileName: 行号（函数名）
  • 设置条件断点 ：
    • b 10 if value==19
  • 删除断点：
    • delete --del -d
    • d 断点的序号
    • 获取编号： info , --i info b
  1. 开始执行gdb调试：
  • 执行一步操作：
  • 继续执行：
  • 执行多步，直接停在断点处：
  1. 单步调试
  • 进入函数体内部： s
  • 从函数体内部跳出：f
  • 不进入到函数内部：n
  • 退出当前循环：u
  1. 查看变量的值： p –print
  2. 查看变量的类型：ptype 变量名
  3. 设置变量的值： set var 变量名=赋值
  4. 设置追踪变量：
  • display 取消追踪变量
  • undisplay 编号
  • 获取编号： info display
  1. 退出gdb调试 quit
