makefile学习

  1. 命名： Makefile/makefile
  2. makefile的三要素：目标、依赖、命令
    • 目标：依赖条件
    • 命令
  • 示例： mycalc add.c sub.c mul.c div.c

         (tab) gcc add.c sub.c mul.c div.c -o mycalc
  1. 如果有多行目标，默认第一行的为终极目标
  2. makefile 中的自动变量:
  • $<:规则中的第一个第一个依赖
  • $@:规则中的目标
  • $^:规则中的所有依赖

    只能在规则的命令中使用

    示例：

    target=app

    $(target):$(obj)
      $(CC) $(obj) -o $(target)<br>
    src=$(wildcard ./*.c)

    obj=$(patsubst ./%.o,./%.c,$(src) )

CC=gcc

%.o:%.c

$(CC) -c $< -o $@

.PHONY:clean

clean:

rm $(obj) $(target) -f

hello:

echo “hello,makefile”

  1. 备注： 如果加上-rm,多了”-“说明如果执行失败继续执行下去
